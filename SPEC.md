## 📌 프로젝트 개요

### 목적
사용자가 일상 속에서 할 일을 쉽게 관리하고 놓치지 않도록 설계된 웹 기반 할 일 관리 애플리케이션입니다. 접근성과 시각적 편안함을 극대화하여 사용자의 생산성을 높입니다.

### 대상 사용자
- 📱 **개인:** 일상적인 할 일 관리가 필요한 사람
- 💼 **직장인:** 프로젝트 작업 관리 및 데드라인 추적
- 👨‍🎓 **학생:** 과제 및 시험 준비 일정 관리
- 🎯 **팀 리더:** 작은 팀의 일일 업무 추적 (향후 공유 기능)

### 핵심 가치
- ✅ **접근성 극대화:** 매번 새로운 창을 열지 않아도 쉽게 접근
- 🎨 **시각적 편안함:** 눈에 피로를 덜어주는 디자인
- ⚡ **빠른 입력:** 3단계로 할 일 추가 완료
- 🔔 **똑똑한 알림:** 마감 기한을 자동으로 추적

## 🎯 주요 기능 목록

### 화면 1: 📋 할 일 목록 (메인 대시보드)

**목적:** 사용자의 모든 할 일을 한눈에 확인

**화면 구성:**
```
┌─────────────────────────────────────┐
│        🎯 StayOn - 할 일 관리      │
│   늘 곁에 있는 할 일 관리 앱        │
├─────────────────────────────────────┤
│ 📋 할 일 목록 | ➕ 할 일 추가 | 📊 통계│
├─────────────────────────────────────┤
│  ⚙️ 사이드바 (필터, 통계)          │
│                                     │
│  [카드1] 프로젝트 계획서 작성       │
│  [체크박스] 완료 | [버튼] 삭제     │
│  ───────────────────────────────────│
│  [카드2] UI 디자인 검토            │
│  [체크박스] 완료 | [버튼] 삭제     │
└─────────────────────────────────────┘
```

**기능:**
1. **할 일 카드 표시**
   - 제목, 설명, 마감 시간, 우선순위 표시
   - 마감 임박(24시간 이내)시 카드 배경색 변경 (분홍색)
   - 진행 시간 실시간 표시 (예: "4분 남음")

2. **상태 관리**
   - ✅ 완료 상태 체크박스
   - 🗑️ 삭제 버튼

3. **필터 (사이드바)**
   - 📋 모든 항목
   - ⏳ 진행 중 (미완료 항목)
   - ✅ 완료됨

4. **정렬 옵션**
   - ⏱️ 마감 기한 순 (긴급 항목 먼저)
   - 🔴 우선순위 순 (High → Medium → Low)

5. **통계 (사이드바)**
   - 전체 항목 수
   - 진행 중 항목 수
   - 완료된 항목 수

---

### 화면 2: ➕ 할 일 추가 (입력 폼)

**목적:** 새로운 할 일을 빠르고 쉽게 추가

**입력 항목:**

| 항목 | 유형 | 예시 | 필수 | 제한 |
|------|------|------|------|------|
| 📝 제목 | 텍스트 | "프로젝트 완료하기" | ✅ | 최대 100자 |
| 📄 설명 | 긴 텍스트 | "마크다운 문서 작성" | ❌ | 무제한 |
| 📅 마감 날짜 | 날짜선택 | 2026-03-25 | ✅ | 현재 이후 |
| ⏰ 마감 시간 | 시간선택 | 18:00 | ✅ | 현재 이후 |
| 🔴 우선순위 | 선택 슬라이더 | Low/Medium/High | ✅ | 3가지 옵션 |

**유효성 검증:**

```
1. 제목 입력 검증
   ❌ 에러: 제목이 비어있음
   ❌ 에러: 제목이 100자 초과
   ✅ 성공: 제목 입력됨

2. 마감 기한 검증
   ❌ 에러: 마감 기한이 현재 시간보다 이전
   ✅ 성공: 마감 기한이 현재 이후
```

**제출 버튼:**
- ✅ 할 일 추가 (성공 시 확인 메시지 표시 후 새로고침)

---

### 화면 3: 📊 통계 및 분석

**목적:** 할 일 관리 현황을 시각적으로 파악

**표시 요소:**

1. **기본 통계**
   ```
   전체 할 일: 5개
   완료된 항목: 2개
   진행 중: 3개
   완료율: 40%
   ```

2. **진행률 바**
   - 시각적 진행 상황 표시
   - 색상: 초록색(진행 중) → 연초록(완료)

3. **우선순위 분포 (막대 그래프)**
   - X축: High, Medium, Low
   - Y축: 개수
   - 색상: Sage Green (#557C55)

4. **마감 임박 경고**
   - 24시간 이내 마감인 항목만 표시
   - 빨간색 경고 아이콘 (🚨)
   - 남은 시간 표시

## 💾 데이터 처리 방식

### 데이터 저장소
- **저장 위치:** Streamlit 세션 상태 (`st.session_state`)
- **데이터 구조:** Python 딕셔너리 리스트

```python
{
    "id": 1,                              # 고유 ID
    "title": "프로젝트 계획서 작성",      # 제목
    "description": "명세서 완성",         # 설명
    "deadline": "2026-03-25 18:00",      # 마감 시간
    "priority": "High",                   # 우선순위
    "completed": False                    # 완료 여부
}
```

### 데이터 생명주기
1. **생성:** 사용자가 할 일 추가 시 새 객체 생성
2. **조회:** 필터/정렬 옵션 선택 시 필터링 수행
3. **수정:** 완료 체크박스로 상태 변경
4. **삭제:** 삭제 버튼 클릭 시 목록에서 제거

### ⚠️ 주의사항
- 현재는 **브라우저 새로고침 시 데이터 초기화**됨
- 향후 데이터베이스 연동 시 영구 저장 가능

## 🖥️ 배포 환경 요구사항

### 필수 환경
- **Python 버전:** 3.11.x 또는 3.12.x
- **OS:** Windows, macOS, Linux 모두 지원

### 필수 패키지

| 패키지 | 버전 | 용도 |
|--------|------|------|
| streamlit | 1.55.0+ | 웹 앱 프레임워크 |
| pandas | 2.2.0+ | 데이터 처리 |
| plotly | 5.18.0+ | 차트 시각화 |

### 설치 방법

```bash
# 1. 가상환경 생성
python -m venv venv

# 2. 가상환경 활성화
# Windows:
venv\Scripts\activate
# macOS/Linux:
source venv/bin/activate

# 3. 패키지 설치
pip install -r requirements.txt

# 4. 앱 실행
streamlit run app.py
```

### 배포 플랫폼

**Streamlit Cloud (무료)**
```
1. GitHub에 코드 push
2. streamlit.io/cloud에서 배포
3. Secrets 탭에서 환경변수 설정 (필요 시)
4. 앱 실행 (자동 HTTPS)
```

**Heroku / AWS / Google Cloud (유료)**
- 향후 공식 문서 참조

## ❓ 자주 묻는 질문 (FAQ)

**Q: 데이터가 저장되나요?**  
A: 현재는 브라우저 새로고침 시 초기화됩니다. 향후 데이터베이스 연동 예정입니다.

**Q: 여러 기기에서 동기화되나요?**  
A: 현재 단일 브라우저만 지원합니다. 클라우드 동기화는 향후 기능입니다.

**Q: 알림이 오나요?**  
A: 현재는 앱 내 경고 표시만 지원합니다. 푸시 알림은 향후 기능입니다.
